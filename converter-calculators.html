Number Base Conversion

What it does: Converts numbers between different number systems (binary, octal, decimal, hexadecimal).

Binary (Base 2): Uses only 0 and 1 (e.g., 1010 = 10 in decimal)

Octal (Base 8): Uses digits 0-7 (e.g., 12 = 10 in decimal)

Decimal (Base 10): Standard number system (e.g., 10 = 10)

Hexadecimal (Base 16): Uses digits 0-9 and A-F (e.g., A = 10 in decimal)

Binary Calculator

What it does: Performs arithmetic operations (addition, subtraction, multiplication, division) on binary numbers.

How to use: Enter binary numbers (using only 0 and 1) and select an operation. The result will be shown in both binary and decimal.

Example: 1010 (binary) + 1100 (binary) = 10110 (binary) = 22 (decimal)

Hexadecimal Calculator

What it does: Performs arithmetic operations on hexadecimal numbers (base 16).

How to use: Enter hex numbers (using digits 0-9 and letters A-F) and select an operation. The result will be shown in both hex and decimal.

Example: A (hex) + B (hex) = 15 (hex) = 21 (decimal)
